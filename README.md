## Aspose.Imaging for .NET

[Aspose.Imaging for .NET](http://www.aspose.com/products/imaging/net) is an advanced image processing API that lets developers create, edit, draw or convert images in their .NET applications. It works independent of other applications and allows saving to Adobe PhotoShopÂ® format (PSD) without having PhotoShop or any other image editor installed on the machine.

This repository contains [Examples](Examples), [Plugins](Plugins) and Showcase projects for [Aspose.Imaging for .NET](http://www.aspose.com/products/imaging/net) to help you learn and write your own applications.
